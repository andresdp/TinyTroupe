# Sample Architecture: E-Commerce Platform

This directory contains a sample architecture description used for testing
and demonstrating the `atamsim` package.

## Files

- `architecture_overview.md` — A description of the system architecture for a
  fictional e-commerce platform ("ShopFlow").
- `business_drivers.md` — Sample business drivers document for the same
  project.

These documents can be passed to `ATAMStakeholderFactory` as grounding
documents and to `ATAMSession` as the architecture context.
